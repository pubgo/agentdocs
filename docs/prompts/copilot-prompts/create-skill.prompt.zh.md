## 对话摘录
首先，查看对话历史记录。如果用户一直遵循多步骤工作流程或方法（例如，调试方法、审查清单、实施模式），请将其概括为可重用的技能。摘录：
- 遵循的分步过程
- 决策点和分支逻辑
- 质量标准或完工检查

## 如果需要请澄清
如果对话中没有出现明确的工作流程，请澄清：
- 这项技能应该产生什么结果？
- 工作空间范围还是个人范围？
- 快速清单还是完整的多步骤工作流程？

## 迭代
1. 起草技能并保存。
2. 找出最模糊或最薄弱的部分并询问这些部分。
3. 一旦最终确定，总结该技能产生的结果，建议尝试该技能的示例提示，并提出下一步要创建的相关定制。

请记住遵循“代理定制”指南来创建高效的技能。
