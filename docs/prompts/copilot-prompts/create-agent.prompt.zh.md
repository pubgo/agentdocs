## 对话摘录
首先，查看对话历史记录。如果用户一直以专门的方式使用代理（例如，限制工具、遵循特定角色、专注于某些文件类型），请将其概括为自定义代理。摘录：
- 所承担的专门角色或角色
- 工具偏好（使用哪些，避免哪些）
- 领域或工作范围

## 如果需要请澄清
如果对话中没有出现明确的专业知识，请澄清：
- 这个代理人应该做什么工作？
- 什么时候应该选择它而不是默认代理？
- 应该使用（或避免）哪些工具？

## 迭代
1. 起草代理文件并保存。
2. 找出最模糊或最薄弱的部分并询问这些部分。
3. 一旦最终确定，总结代理的作用，建议尝试的示例提示，并提出下一步要创建的相关定制。

请记住遵循“代理定制”指南来创建高效的代理。
