# 建议修复问题

## 目的

根据给定问题的详细信息提出潜在的解决方案。此技能分析问题的内容，包括其描述、评论和任何相关上下文，以生成有助于有效解决问题的建议。

## 用法

要使用此技能，请提供问题的详细信息，包括其描述、评论和任何相关上下文。然后，该技能将分析信息并生成如何解决问题的建议。这对于寻求如何解决问题的指导的开发人员或刚开始解决问题的开发人员来说特别有用。

## 如何提出修复建议

- 仔细阅读问题的描述和评论，以充分理解问题。
- 在可能的情况下输出代码块并参考工作区中的实际文件进行修复。
- 如果问题与错误相关，请尝试找出根本原因并提出直接解决问题的修复方案。
